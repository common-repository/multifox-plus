== Description ==

Multifox Plus plugin adds additonal features for [Multifox](https://wordpress.org/themes/multifox/). Some of the features are,

1. Custom Header - unlimited possiblities
2. Custom Footer - unlimited possiblities
3. Breadcrumb customization
4. Typography -for body, heading tags,...
5. Custom blog listing styles
6. Custom hooks - to place your script on any part of the site
7. Site layout - fullwidth & boxed
8. Custom skin - customize your site for any color tone
9. Elementor shortcodes
10. Other features - to top, loader,...

== INSTALL THE MULTIFOX PLUS MANUALLY ==

1. From the dashboard of your site, navigate to Plugins â€“> Add New.
2. Select the Upload option and hit 'Choose File'.
3. When the popup appears, select the multifox-plus.zip file from your PC.
4. Activate the plugin through the 'Plugins' menu in WordPress.
